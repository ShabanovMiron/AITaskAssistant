# AI Task Assistant

Веб-приложение для управления задачами с авто-определением приоритета и категории по тексту.

## Запуск

### 1. База данных PostgreSQL

Создайте БД: `tasksdb`
Выполните `sql/init.sql` (через pgAdmin Query Tool)

### 2. Backend (Node.js)

cd backend
npm install
скопировать .env.example в .env и заполнить
npm run dev

### 3. Python-сервис

cd python-service
python -m venv venv
активировать: venv\Scripts\activate
pip install -r requirements.txt
python app.py

### 4. Frontend

cd frontend
npm install
npm start

## Проверка

Открыть http://localhost:3000
Зарегистрироваться / войти.
В дашборде создавайте задачи – приоритет и категория будут определены автоматически.
Вы можете редактировать (изменять статус, приоритет, категорию, название, описание) и удалять свои задачи.
Все задачи видны только их создателю.

## Список ключевых слов для расставления приоритета и категории:

### Приоритет

high (срочно): срочно, пятница, сегодня, аврал
medium (средний): важно, скоро, завтра
low (низкий): остальные случаи

### Категория

business (бизнес): презентация, клиент, встреча, переговоры
analytics (аналитика): отчет, анализ, исследование
documentation (документы): документ, письмо, контракт, договор
general (общее): всё остальное

При недоступности Python-сервиса используются значения по умолчанию: medium, general
